# 基本语法概述

基础语法包括：

- 变量，脚本变量的定义以及使用说明，包括数字，字符串，结构体等；
- 数据集，软件的数据集操作指令；
- 程序控制语句，包括判断和循环等；
- 函数，自定义函数的使用和说明；
- 保存、导入、导出，脚本初始化，数据的保存、导入和导出。
